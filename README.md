* thinkphp5.0 中文最佳实践PHP开源框架 http://www.thinkphp.cn/
* layui 由职业前端倾情打造，面向所有层次的前后端程序猿，零门槛开箱即用的前端UI解决方案 http://www.layui.com/
* 后台使用  jqadmin  https://jqadmin.jqcool.net/
* 在此感谢被1kbcms所使用的各种组件
